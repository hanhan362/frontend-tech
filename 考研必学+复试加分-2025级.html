第一部分：考研初试必须学好的科目（408 + 数学一 + 英语一 + 政治）

1. 数学（一）—— 150 分，拉分王

#科目学分何时学要求
1高等数学（上）5大一下极限、导数、积分、微分方程——考研占比 56%
2高等数学（下）5大二上多元微积分、级数、曲线曲面积分——考研占比 22%
3线性代数3大二上行列式、矩阵、向量、特征值——考研占比 22%
4概率论与数理统计3大二下概率、分布、估计、检验——数一专属
⚠️ 你挂的如果是高数，这是最要命的一门。高数上下共 10 学分，考研数学一 150 分中高数占约 80 分。重修必须扎扎实实搞懂，这不是"过了就行"，是"考研命脉"。

2. 408 计算机学科专业基础 —— 150 分

#科目学分何时学说明
1数据结构4大二上408 中占比最高（~45 分），你学的是英文版，必须刷中文王道/天勤
2计算机组成原理4大二下408 最难（~45 分），英文教材 + 中文辅导书双管齐下
3操作系统4大三上~35 分，进程管理、内存管理、文件系统
4计算机网络4大三上~25 分，五层模型、TCP/IP、HTTP
💡 408 平均分常年 60-70。你的优势：四门课在课堂上系统学过（虽然是英文）。劣势：考研是中文卷。对策：上课期间就把王道/天勤当参考书同步看，别等到大三才开始。

3. 英语（一）—— 100 分

#能力要求
1词汇5500+ 大纲词，重点 2000 核心词
2阅读4 篇 × 10 题 = 40 分，决定总分
3作文大作文 20 + 小作文 10 = 30 分
4完形 + 翻译 + 新题型各 10 分
中外的优势：你日常用英语上课，阅读能力应该比普通软工强。考完雅思再考英语一，阅读部分基本降维打击。但考研英语有自己独特的出题套路，真题必须刷 2 遍以上。

4. 政治 —— 100 分

大三暑假开始即可，现在完全不急。选择题 50 + 分析题 50，肖秀荣 1000 题 + 肖四肖八是标配。

第二部分：复试加分项

你有挂科记录，复试时导师一定会注意到成绩单上的污点。下面的加分项，每一个都是在帮你"擦掉"这个污点。

⭐ 硬通货加分（含金量从高到低）

等级加分项获得难度何时开始详细说明
S 级 发表论文（SCI/EI/CCF） 极高 大二联系导师 找学院老师进实验室，本科生一作或二作发论文，复试直接碾压大多数人
S 级 ACM/ICPC 区域赛银牌以上 极高 大一暑假开始刷题 算法竞赛顶级奖项，985 导师抢着要
A 级 蓝桥杯国赛一等奖/二等奖 高 大二开始 比 ACM 简单，性价比高，每年 4 月初赛
A 级 国家级大创项目（优秀结题） 中 大二申请 大学生创新创业训练计划，每年申报，跟导师做一年
A 级 专利/软著（发明专利最值钱） 中 大二-大三 配合项目产出，实用型专利 + 软件著作权
B 级 GitHub 高质量开源项目（100+ stars） 中 现在开始 持续维护一个项目，网站/工具/组件都可，复试拿出来直接展示
B 级 雅思 7.0+ 中 大二暑假 中外办学的你考这个是刚需，顺便复试也能用
B 级 六级 600+ 中 大二 比雅思门槛低，但 600 分以上导师会多看一眼
C 级 高含金量实习（大厂核心岗） 中 大二-大三暑假 字节/腾讯/阿里/美团后端或算法实习，不是打杂岗
C 级 竞赛省奖（蓝桥杯省一、数学建模省一+） 中低 大二 门槛低，人人可冲，有比没有强

复试时导师最看重的（按优先级）

1. 科研能力 → 论文 / 大创 / 实验室经历
2. 编码能力 → ACM / 蓝桥杯 / GitHub 项目 / 机试成绩
3. 英语能力 → 雅思 7.0+ / 六级 600+ / 英文论文
4. GPA 趋势 → 大一差但后续一路涨 > 全程平庸
5. 实习经历 → 大厂核心岗有意义，水实习不如没有

第四部分：你的最低加分目标（挂科生的及格线）

因为你有挂科，复试时至少要拿到以下 3 个加分项中的 2 个，否则导师大概率会 pass 你：
#目标交付物
1 雅思 7.0 成绩单，顺便满足外方学位要求
2 蓝桥杯国赛二等奖以上 获奖证书，证明编码能力
3 国家级大创 + 一篇论文 结题证书 + 录用通知
底线：这三件事，大二结束前至少启动两件。你比别人少了一样东西（完美的成绩单），就得用别的东西补回来。这不是"要不要做"，是"必须做"。
